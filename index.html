Skills

Hello, I am Sean Dove...

I am a passionate front end web developer with persistence and dedication. Problem solving, creating, building, and technology are what I love. Fabricating something from nothing, and expanding and collaborating with others in an innovative environment is an exciting and amazing feeling. With over two decades as a musician and a decade of experience being a luthier, I found myself dreaming of my own online business selling guitars. In researching how to build my own website, I came to the realization that my true passion lies in creating, whether it be guitars or websites. This lead me to dive head first into web development. My curiosity has always fueled my problem solving skills, which has carried over from building and repairing guitars to web development. It's something I enjoy so much it never feels like work. I'm looking for the right opportunity to join a team with other individuals who share the same passion as I do in the field and to expand my knowledge.

Applications

Fridge 2 Plate

Fridge 2 Plate is an application that allows users to look up recipes based on ingredients they already have on hand as well as post their own recipes to be featured.

Details:
  • Filters recipes by ingredients
  • Allows users to add their own recipes to be featured on the front page
  • Integrates user sign-in using Google O-Auth and payment processing with BrainTree
  • Recipe sharing with social media
  • My involvement was working in a team environment and implementing agile and scrum methodology, and developing front-end functionality for ingredient filter buttons

Technologies Used: LAMP stack (Linux, Apache, MySQL and PHP), APIs (BrainTree, Google OAuth, Facebook OAuth), JavaScript, jQuery, jQuery UI, Bootstrap, HTML5 & CSS3, Git, GitHub

Casino Tic-Tac-Toe

A Casino themed Tic-Tac-Toe game with variable game board sizes and an option for Ultimate Tic-Tac-Toe.

Details:
  • Variable game board size from 3 x 3 up to 10 x 10 grid.
  • Responsible for wire framing layout and over all look, feel, and styling of the game. Also implemented functionality for stats, and Ultimate Tic-Tac-Toe.
  • Worked with in a team utilizing agile and scrum methodology.

Technologies Used: HTML5 & CSS3, JavaScript, jQuery, Bootstrap, Git, GitHub

Ever Hungry?

A mobile application utilizing multiple API's to help you decide on what to eat and locations of restaurants near you that you can eat at or pick up from.

Details:
  • Already had a hamburger for lunch and don't want it in your search for dinner? User can un-select certain foods they wish not to have show up in the search.
  • Uses GPS location or zip code to help find food near you.
  • Google places API integrated to find compatible restaurants.
  • Google maps API integrated to get directions
  • Flickr API integrated to provide related photos.
  • Implemented agile and scrum methodology in a team environment.
  • My involvement included implementing Flickr API and modal styling as well as help with over all application layout and styling.

Technologies Used: HTML5 & CSS3, JavaScript, jQuery, Google Places API, Google Maps API. Flickr API, Git, GitHub
